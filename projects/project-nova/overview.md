# project NOVA

## 목표

> 클라이언트 메일의 요청사항을 AI가 분해해 프로젝트별로 분류·이슈 등록까지 자동화하는 Skill을 구현해 사내 공모전(Project NOVA)에 제출한다.

## 공모전 일정

| 항목 | 일자 |
|------|------|
| 팀 구성 및 주제 확정 | 2026-05-18 |
| 접수 문서 최종안 회의 | 2026-05-20(수) |
| 접수 문서 제출 | 2026-05-22(금) |

## 팀

- 전채린 외 4명 (총 5명 공동 출전)

## 기술 스택

| 영역 | 기술 | 선택 근거 |
|------|------|----------|
| AI | 사내 LLM | 내부 메일 보안·데이터 주권 |
| 오케스트레이션 | n8n | 시각적 워크플로우, 빠른 구축 |
| 메일 연동 | 사내 메일 시스템 (API 또는 IMAP) | 다우오피스 연동 가능 여부 확인 필요; 불가 시 직접 업로드 방식 fallback |
| Git 연동 | GitLab API / GitHub REST API | 이슈 자동 생성 (아웃풋 형태 미확정) |
| 첨부파일 파싱 | 별도 파싱 모듈 검토 중 | PDF·엑셀·PPT 지원 필요 |
